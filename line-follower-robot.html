Line Follower Robot | Discrete Combinational Logic

1. Project Overview

The Line Follower Robot is an autonomous vehicle designed to navigate a path using hardwired combinational logic. By eliminating the need for a microcontroller, the system achieves near-instantaneous processing of sensor data. The robot utilizes a three-sensor array and discrete logic gate ICs (NAND/NOT) to drive an H-Bridge motor driver for real-time path correction.

  • Platform: Discrete Logic Control (No Microcontroller)
  • Role: Logic Designer & Hardware Systems Developer
  • Key Components: Logic Gate ICs (74-series), IR Sensors, H-Bridge Motor Driver, DC Motors.

2. Core Features

  • Hardwired Decision Making: Pathfinding is executed through physical logic gates, implementing a custom Boolean algorithm derived from K-map simplification.
  • NAND-Based Architecture: Uses universal NAND gates to implement the control functions $X = A' + B$ and $Y = B + C'$, optimizing the physical IC footprint on the PCB.
  • Real-Time H-Bridge Control: Directly interfaces logic signals with an H-Bridge motor driver to regulate power to the Left and Right motors.
  • Zero-Latency Navigation: Processing speed is limited only by the propagation delay of the logic gates (nanoseconds), ensuring immediate response to track curves.
