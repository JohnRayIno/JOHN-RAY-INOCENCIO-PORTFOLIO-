Numerical Methods Calculator

Engineering Computation | Java Software Development

1. PROJECT OVERVIEW

The Numerical Methods Calculator desktop application designed to solve advanced mathematical problems through iterative algorithms. This project focuses on providing a reliable table interface for root-finding, linear system solving, and numerical integration through engineering math and functional software.

Platform: Windows

Role: Software Developer

Tools: Java, Java Swing (GUI), exp4j (Mathematical Expression Parsing)

2. CORE FEATURES

  • Root Discovery Engine: Implements the Bisection and Secant methods to find the roots of non-linear equations ($f(x) = 0$) that are otherwise impossible to solve algebraically.
  • Linear Systems Solver: Features Doolittle (LU) Decomposition and the Gauss-Seidel iterative method. These allow for the solving of large systems of linear equations ($Ax = b$), essential for circuit analysis and structural engineering.
  • Numerical Integration: Utilizes Simpson’s 1/3 Rule to approximate the area under a curve. By using parabolic interpolation, it provides high-accuracy results for work, energy, and volume calculations.
  • Step-by-Step Iteration Tracking: Unlike standard calculators, this suite displays a complete iteration table (as seen in the GUI screenshots), showing the midpoint, function values, and error percentage at every single step.
  • Precision Control: Integrated a dynamic rounding utility that allows users to specify decimal precision, simulating how different hardware architectures handle floating-point calculations.
