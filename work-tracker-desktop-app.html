Work Tracker Desktop App

Desktop Application | Productivity & Task Management

1. Project Overview

WorkTracker is a comprehensive desktop productivity suite designed to streamline task management through data-driven insights and gamified progress tracking. The application integrates a robust SQL-based backend with a modern graphical interface to help users organize, time, and analyze their professional workflows effectively.

Platform: Windows / macOS / Linux (Python Desktop Application)

Role: Lead Developer

Tools Used: Python, SQLite, ttkbootstrap, Matplotlib, Pandas, Pillow

2. Core Features

  • Gamified Progression System: A dynamic rewards engine that awards 10 EXP per completed task. The system calculates user levels (1–30) in real time and unlocks new profile avatars as milestones are reached.
  • Intelligent Task Management: A centralized hub for creating and tracking tasks with metadata such as priority level, category, and estimated duration. Includes automatic overdue task detection based on system dates.
  • Advanced Analytics Dashboard: Data visualization using Matplotlib and Pandas to generate monthly productivity trends and task distribution charts.
  • Precision Productivity Timer: A built-in stopwatch that tracks active work sessions to measure how long projects take to complete.
  • Secure Session Management: Custom authentication system with persistent login sessions and secure database interactions to protect user data.

3. Design & Logic

User Experience

The interface follows the "Flatly" design system from ttkbootstrap to maintain a clean and professional layout. The application prioritizes a zero-friction workflow where dashboards and task tables automatically refresh whenever the database is updated. A fade-in splash screen initializes the SQLite database and validates user sessions before loading the main workspace.

Database Architecture

The application uses a relational database structure composed of three main tables: users, tasks, and rewards. This architecture maintains data integrity while enabling efficient analytics queries such as monthly productivity reports.
