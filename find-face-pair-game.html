Find Face Pair Game

Mobile Game Development | UI/UX Design

1. PROJECT OVERVIEW

Find Face Pair is a polished Android memory game where players race against time to match pairs of character cards. This project highlights the integration of smooth user interfaces with logical backend systems, focusing on creating a fun and engaging mobile experience.

Platform: Android (Landscape)

Role: Team Developer (Team of 3)

Tools: Kotlin, Jetpack Compose, Android Studio

2. CORE FEATURES

  • Dynamic Matching System: A 6x6 grid that shuffles 36 cards every session to ensure a unique experience every time.
  • Difficulty Scaling: Integrated three distinct modes (Easy / *, Medium / **, Hard / ***) that adjust the countdown timer (10min / 5min/ 3min) to challenge different player skill levels.
  • Synchronized Multimedia: A centralized audio system that manages background music and sound effects, ensuring the audio reacts instantly to game events like winning or losing.
  • Arcade-Style UI: Optimized for landscape orientation with custom typography and vibrant color-coded feedback.

3. DESIGN & LOGIC

The User Experience: We designed the interface to be intuitive. When two cards are flipped, the system provides a brief one-second window—allowing the player to memorize the images—before automatically flipping them back. To ensure a bug-free experience, the board "locks" during this validation phase so players cannot accidentally break the game logic by tapping too many cards at once.
